## T2. 언어 전환 인터랙션 완전 제거 [실측 원인 확정]
실측 결과 page-fade, duration-reveal, transition-[transform,opacity] 클래스가 언어 전환 시에도 발동해 "띠용"하고 다시 나타난다. 원인: 언어(LangContext) 변경 시 컴포넌트가 리마운트되거나 리빌/페이지 전환 애니메이션이 재트리거된다. 수정:
1. 언어 전환은 리마운트 없이 텍스트 노드만 교체되게 하라. LangContext 값 변경이 페이지/섹션 컴포넌트의 key를 바꾸거나 언마운트→재마운트를 유발하지 않도록(같은 컴포넌트 인스턴스 유지, 자식 텍스트만 갱신).
2. 등장 애니메이션(page-fade, duration-reveal, Reveal/useReveal)이 "최초 마운트 시 1회"만 실행되고 언어 전환으로는 절대 재실행되지 않게 하라(이미 표시된 요소는 재애니메이션 금지).
3. 언어 전환 시 허용되는 변화는 텍스트의 opacity 크로스페이드뿐. transform 이동·slide·재등장 전부 금지. 레이아웃 시프트 0.
4. 전 페이지에서 언어 토글을 눌러 어떤 섹션도 다시 슬라이드·페이드인 되지 않는지 검증하라.
