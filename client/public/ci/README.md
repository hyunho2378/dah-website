# /ci/ 정적 에셋 슬롯 (26_CI_PAGE)

CI 페이지(/about/ci)가 참조하는 정적 파일 위치. 파일을 이 폴더에 넣으면 즉시 반영된다.
파일이 없으면 다운로드 버튼은 "준비 중" 비활성, 이미지 슬롯은 "이미지 준비 중" 플레이스홀더로 표시된다.

## 다운로드 (CI 의미 섹션)
- `dah-ci-manual.pdf` — 매뉴얼 다운로드
- `dah-ci.jpg` — JPG 다운로드
- `dah-ci.ai` — AI 다운로드

## 이미지 슬롯
- `symbol.png` — 대표 심벌
- `logo-kr.png` — 한글 가로형 로고
- `logo-en.png` — 영문 가로형 로고
- `signature-vertical.png` — 시그니처 상하조합형
- `signature-horizontal.png` — 시그니처 좌우조합형
- `motif.png` — 그래픽모티브

경로·파일명은 어드민(/admin/ci)에서 개별 교체·업로드도 가능하다.
