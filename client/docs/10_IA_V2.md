# 10_IA_V2 — 정보 구조 재설계 (기존 IA.md 대체)

## 0. 재설계 판정

구글 사이트 IA의 문제 3개를 이렇게 푼다.
1. About과 전공소개 중복 → 하나로 통합. 최상위 메뉴명은 "About"(국문에서도 About 유지, 사용자 확정)
2. 연혁 독립 페이지 → About 하위 섹션으로 흡수. 연혁만 보러 오는 방문자는 없다
3. 포트폴리오·취업현황 산재 → "진로" 한 페이지로 통합. 학생 성과는 독립 승격 + 전용 디자인(성좌 컨셉)

트랙명 확정: 디자인 트랙 / AI 트랙 / 엔터컬쳐 트랙 (구 미래융합디자인·AI디지털인문학·문화예술콘텐츠)

## 1. 헤더 (KPC 문법 이식)

```
[로고 SVG]   About   교육과정   프로그램   학생   쇼케이스   소식   |   EN   로그인
```
- 데스크탑: 호버 시 풀폭 메가메뉴(하위 페이지 그리드). KPC 방식
- 로그인 상태별: 비로그인=로그인 버튼만 / 로그인=역할 뱃지+관리 진입. 비로그인 화면에는 편집 UI 픽셀 하나도 노출 금지
- 모바일: 햄버거 금지. 하단 GlassDock(리퀴드 글래스 필, 탭 시 다이나믹 아일랜드처럼 확장) — DESIGN_V2 참조
- 전시회 접수 기간에는 헤더 우측에 "접수하기" 버튼 노출(관리자가 site_settings로 on/off, 위치 선택)

## 2. 사이트맵 v2

```
/                      홈
/about                 전공 소개 (개요 + 미션·비전 + 연혁 타임라인 + 대학원 안내)
/about/people          교수진 · 멘토단
/curriculum            교육과정 (트랙 3 + 로드맵 다이어그램)
/curriculum/codesharing 코드쉐어링 (신청서 HWP 다운로드 포함)
/programs/exhibitions  전시회 아카이브 (포스터 그리드, 2017~)
/programs/exhibitions/:id  전시회 상세 (좌 포스터 / 우 정보 / 하단 현장·작품 갤러리)
/programs/contests     공모전 (포스터 그리드)
/programs/contests/:id 공모전 상세 (외부 접수 링크 분기 가능)
/programs/lectures     특강
/programs/lectures/:id 특강 상세
/students/council      운영위원회 (기수별 아카이브: 로고·이름·소개·구성원, 동일 템플릿 보존)
/students/clubs        동아리
/students/achievements 학생 성과 (전용 성좌 디자인, 일반 게시판 금지)
/students/careers      진로 (취업 현황 + 재학생 포트폴리오 통합)
/showcase              웹&앱 쇼케이스 (그리드)
/showcase/:id          쇼케이스 상세 (메인 1 + 서브 2, 16:9 고정)
/showcase/submit       쇼케이스 제출 (비로그인 공개, 항목 고정 폼)
/news                  공지사항 (대내/대외/공모전모집/특강모집 태그 필터, KPC 게시판 문법)
/news/:id              공지 상세 (리치 콘텐츠 + 공유 버튼)
/resources             자료실
/submit                전시회 접수 (기간 한정 활성)
/submit/edit           접수 수정 (이메일+비밀번호 인증)
/login                 로그인 (최초 접속 시 비밀번호 설정 플로우)
/admin/*               관리 대시보드 (권한별)
/en/*                  영문 미러 (전 공개 페이지 1:1, 전문 번역)
*                      404
```

## 3. 갤러리 처리 (더보기 폐기)

- 포토·영상 갤러리 독립 메뉴 폐기. 사진·영상은 각 행사 상세 하단 갤러리로 귀속
- 행사 귀속이 애매한 잔여 미디어만 /resources 하위 갤러리 탭
- "더보기" 메뉴는 존재 자체를 금지

## 5. 페이지 템플릿 계약 (KPC 서브페이지 문법)

모든 서브페이지 공통: PageBanner(페이지명 + 브레드크럼 + 우주 배경 변주) → 탭/사이드 네비(하위 있을 때) → 콘텐츠 → 공유 버튼(상세 페이지). 상세 페이지 템플릿 4종:
- T1 게시글형: 공지·자료실 (리치 콘텐츠)
- T2 포스터형: 전시회·공모전·특강 (좌 포스터 2:3, 우 메타정보 표, 하단 갤러리·외부 링크)
- T3 쇼케이스형: 상단 메인 이미지 16:9, 메타(제목/주제/팀·개인/설명/활용 툴/링크), 서브 이미지 2
- T4 아카이브형: 운영위원회 기수 페이지 (로고, 기수명, 소개, 구성원 그리드)

모든 리스트·상세·수정 가능 콘텐츠는 CMS_SPEC의 편집 매트릭스를 따른다.
